# 2gather-web

A premium, glassmorphic collaboration platform built with React and Vite.

## Features
- **Cinematic Experience**: Immersive UI with glassmorphism and ambient design.
- **Dynamic Galleries**: Dome and Circular galleries for interactive visual storytelling.
- **Responsive Design**: Optimized for various screen sizes.
- **State-of-the-art Styling**: Custom CSS with modern typography and animations.

## Tech Stack
- React
- Vite
- Framer Motion (for animations)
- Vanilla CSS

## Development
```bash
npm run dev
```
